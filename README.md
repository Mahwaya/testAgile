# WSBAgile - SPA for Agile Project Management

Single Page Application for a small menu browser, created by an Agile student team.

The app shows a menu for a coffee shop with three main sections:
Starters, Main, Desserts.
Users pick a category and see a list of dishes, then open a detail view.

---

## Project goals

- Practice Agile work in a small front end project.
- Keep clear roles: UI, Data, JavaScript, Testing and Documentation.
- Build a simple SPA without frameworks.
- Show progress per week in Git and Jira.

---

## Tech stack

- HTML5 for structure  
- CSS3 for layout and styles  
- Vanilla JavaScript for SPA behavior  
- JSON for menu data  
- Git and GitHub for version control  
- Jira board for planning
